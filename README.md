# fullstack_developer_capstone

## Project Overview
This is a full-stack web application for a car dealership. It allows users to browse dealerships, read and submit reviews, and manage their accounts.

## Technologies Used
- **Frontend:** React.js
- **Backend:** Django (Python)
- **Database:** MongoDB (for dealerships and reviews), SQLite (for car models)
- **Microservices:** IBM Code Engine for Sentiment Analysis
- **Containerization:** Docker
- **Orchestration:** Kubernetes

## Features
- User authentication (login, logout, registration)
- View dealerships by state
- Read dealer reviews with sentiment analysis
- Submit new reviews
- Browse car makes and models

## Setup Instructions
1. Clone the repository
2. Set up the backend services
3. Run the Django server
4. Start the React frontend
